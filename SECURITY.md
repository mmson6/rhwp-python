# Security Policy

## Supported Versions

`rhwp-python` is pre-1.0. Only the latest released minor receives security
updates.

| Version | Status             |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |
| < 0.3   | :x:                |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security reports.**

Report privately via GitHub Security Advisories:
<https://github.com/DanMeon/rhwp-python/security/advisories/new>

If you cannot use GitHub, email **jgy5262@gmail.com** instead.

We aim to acknowledge reports within 72 hours and to ship a fix or
mitigation within 30 days. Coordinated disclosure is preferred.

For vulnerabilities in the upstream `rhwp` Rust core (parser, renderer),
please report to <https://github.com/edwardkim/rhwp/issues>.
